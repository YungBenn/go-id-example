# ID Generator

This project generates various types of unique identifiers.

## Types of IDs Generated

1. NanoID
2. CUID
3. ULID
4. UUIDv4
5. UUIDv7

-   UUIDv4: 393a5b73-7c01-4551-95fd-42881181eb67
